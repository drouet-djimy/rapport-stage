Mon travail en cuisine

les différents postes

L'initiateur

L'initiateur est la personne qui va choisir le pain ou la galette (wrap) adapté au sandwich à préparer. Il va ensuite le mettre à chauffer dans le toaster (pour tous les pains) ou dans le steamer (pour les galettes et pain à filet o fish).

toaster

steamer (cuiseur à vapeur)

L'assembleur

L'assembleur se trouve juste après l'initiateur. Il à pour but de récupérer les pains (et les galettes) cuits(es) et de mettre les ingrédients manquants (oignons, salade,tomates,cornichons,...). Il cède ensuite le sandwich à la "dépose protéine" .

table à garniture

La dépose protéïne

La dépose protéine (ou préposer U.H.C. Upright Holding Cabinet.Tiroirs chauffés dans lequel sont conservé les protéïnes) est le poste qui consiste à déposer la protéïne adaptée dans le sandwich. Une fois la protéïne dans le sandwich, il est chargé d'emballer celui-ci, puis de l'envoyer.

UHC

Le Grill

Les Grills sont les plaques sur les-quelles sont chauffées les viandes à plus de 70°C dans le but d'éliminer toutes les bactéries. Les viandes sont cuites en 60 secondes pour les petites viandes et 120 secondes pour les "grandes"

Grills

La ligne frit

La ligne frit est l'endroit ou se trouvent les friteuses de la cuisine. Ce sont dans ces friteuses que sont cuits les poulets, nuggets, mais également les galettes de légumes, les palet de cherve ainsi que le poisson. Les protéïnes sont cuites entre 3 minutes 14 et 5 minutes suivant le produit

Les rêgles en cuisine

Les rêgles dans la cuisine sont strictes pour des raisons de sécurité allimentaire.

  • Toutes les heures tout les équipiers (personnels) doivent aller se laver les mains.
  • Toutes les heures tout les gants grills (gants servant à manipuler les viandes) sont changés.
  • Les chiffons sont également changées toutes les heures pour éviter toute contamination.
  • Le port d'une charlotte et d'un tablier sont obligatoire ainsi que le cache barbe.
  • Chaque action "contaminante" (se toucher le visage, se gratter, toucher ses cheveux,...) entraîne un lavage des mains immédiat.
  • Il est également interdit de courir dans le restaurant pour des raisons de sécurité, les équipiers peuvent se brûler ou entraîner la chute d'objet lourds
